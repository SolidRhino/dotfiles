# Dotfiles Site

This directory contains the Astro + Starlight site for this dotfiles repository.

## Purpose

- publish the main docs site for the repo
- render content from the repository in a browsable format
- deploy to GitHub Pages

## Common commands

Run these from `site/`:

| Command | Action |
| :-- | :-- |
| `npm install` | Install site dependencies |
| `npm run dev` | Start the local dev server |
| `npm run build` | Build the production site |
| `npm run preview` | Preview the built site locally |

## Notes

- Main project documentation lives at the repository root.
- Site-specific config lives in this directory.
- If the site structure or workflow changes, keep this README aligned with the repo's actual setup.
